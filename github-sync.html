    配置 GitHub Token

    Token 仅存储在本地浏览器，不会上传到任何服务器

    • 打开 GitHub → 创建新 Token（点击打开） （需要登录 GitHub 账号）
    • 在 Note 填写名称（如 lauyiu-sync）， 勾选 repo 权限，设置有效期（建议 90 天或无限期）
    • 点击页面最底部 Generate token 按钮，复制生成的 Token
    • 将 Token 粘贴到下方输入框，点击保存并验证

    Token 以 ghp_ 或 github_pat_ 开头，需要 repo 权限
